# Style Target

The RJ-1 asset set should look like a premium industrial energy miniature for an exhibition booth.

- Big World first viewport must show HQ in the center and six obvious selectable landmarks.
- Use clean building masses, glass, solar fields, power equipment, storage racks, drones, vehicles, and service roads.
- Keep forms readable from a 3/4 aerial camera.
- Use cyan, white, gold, and green status accents sparingly on top of neutral enterprise materials.
- Avoid toy-like color blocking and empty flat pads.
